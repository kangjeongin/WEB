< 인터넷과 웹 >

웹 = 월드 와이드 웹
 : 인터넷에 연결된 컴퓨터를 통해 사람들이 정보 공유하는 셰게적 정보 공간

인터넷 >(부분집합기호) 웹 , 이메일, FTP 등등(서비스라고 말하네)




인터넷 동작시키기 위해서는 최소 컴퓨터 2개 필요

웹 브라우저 깔린 컴퓨터(web client, game server 등)랑 
웹 서버깔린 컴퓨터(web server, game server 등)를 인터넷으로 연결

request, response





내가 쓴 거 세계적으로 공유하도록 웹 서버 배우는 것

대신 웹서버 제공 = web hosting  : github

-> settings에서 github pages에서 source를 master branch로 하면
웹 (.html 파일들만 )사이트 주소를 준다


2)
내컴퓨터에 웹서버 설치해서 누구나 내 컴퓨터에 있는 웹페이지를 볼 수 있도록

apache(bitnami 사용), nginx 


+) html파일 만들어서 폰으로 ip주소만 입력하면 폰에서도 내가 만든 웹사이트 열 수 있는거네
